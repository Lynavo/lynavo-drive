# Lynavo Drive v1.0.1

## Choose an edition

### Open Source Edition

The files attached to this GitHub Release are unsigned OSS build-verification
outputs produced from the `v1.0.1` source tag. They are not an official signed
distribution.

| Platform | File | Notes |
| --- | --- | --- |
| macOS Apple Silicon | `LynavoDriveDemo-1.0.1-macos-arm64.dmg` | Unsigned verification build |
| macOS Intel | `LynavoDriveDemo-1.0.1-macos-x64.dmg` | Unsigned verification build |
| Windows x64 | `LynavoDriveDemo-1.0.1-windows-x64.exe` | Unsigned NSIS installer |
| Windows x64 | `LynavoDriveDemo-1.0.1-windows-x64.zip` | Unsigned archive |
| Android arm64 / x86_64 | `LynavoDriveDemo-1.0.1-android-arm64-x86_64.apk` | Unsigned build-verification package; installation is not guaranteed |
| Android arm64 / x86_64 | `LynavoDriveDemo-1.0.1-android-arm64-x86_64.aab` | Build-verification bundle, not directly installable |
| Checksums | `SHA256SUMS` | SHA-256 entries for all six packages |

macOS Gatekeeper, Windows SmartScreen, and Android sideloading controls may
block installation or display warnings. Verify the matching entry in
[`SHA256SUMS`](https://github.com/Lynavo/lynavo-drive/releases/download/v1.0.1/SHA256SUMS)
before testing an asset. The OSS iOS app remains source-build only and is not
attached as an IPA.

### Official / Commercial Edition

For signed desktop installers, official mobile builds, and services available
for the selected plan, use the appropriate official channel:

- [Lynavo Drive official website](https://drive.lynavo.io/)
- [Download for iPhone and iPad on the App Store](https://apps.apple.com/us/app/lynavodrive/id6771620902)
- [Download for Android on Google Play](https://play.google.com/store/apps/details?id=com.vividrop.mobile.global)

Official binaries are built, hosted, licensed, and supported outside this
repository. Their version numbers may differ from `v1.0.1`, and they are not
included in this GitHub Release's assets. Refer to the official website for
current commercial license, pricing, capability, and support terms.
